## 1. 시장 정의 및 범위

### 1.1 시장 경계

**포함**:
- 거래 주체: B2C, B2B, B2G
- 연령: 만 19세 이상 (대학생 포함)
- 언어: 모든 외국어
- 형태: 온라인, 오프라인, 하이브리드, 독학

**제외**:
- 초중고 사교육
- 외국인 대상 한국어 교육
- 해외 어학연수 (현지 수강료)
- 대학 정규 학비 내 어학 수업

### 1.2 Needs 분류

**A. 경제활동** (50-60%):
- 직무 수행, 취업/승진, 시험 준비, 학업/연구

**B. 생활/문화** (30-40%):
- 여행, 미디어 소비, 이민/장기 거주, 인간관계

**C. 자기계발** (5-10%):
- 교양/취미, 개인 목표

**D. 조직** (10-15%):
- 기업 교육, 정부/공공기관

### 4.1 BM별 가치사슬 패턴

#### 오프라인 학원
```
[학습자] → [학원] → [강사 40-50%] + [교재 10-15%] + [임대료 20-30%]
          (100%)      순이익 10-20%
```

**특징**:
- 선불 수강권 → 현금흐름 우위
- 강사 인건비 비중 높음
- 주요 상권 임대료 부담

---

#### 화상 영어
```
[학습자] → [업체] → [튜터 30-40%] + [플랫폼 20-30%] + [마케팅 20-30%]
          (100%)     순이익 10-20%
```

**특징**:
- 해외 튜터 (필리핀, 미국 등)
- 환율 영향
- 마케팅 비용 높음

---

#### 온라인 강의
```
[학습자] → [업체] → [제작 20-30%] + [마케팅 30-40%] + [운영 10-20%]
          (100%)     순이익 10-20%
```

**특징**:
- 마케팅 비용 매우 높음 (TV 광고 등)
- 스케일 확보 시 마진 개선

---

#### 어학 앱
```
[학습자] → [앱] → [AI 개발 40-50%] + [마케팅 30-40%] + [운영 10-20%]

```

**특징**:
- AI 개발 비용 높음
