# 세션 요약 아카이브

이 폴더는 CMIS 개발 세션의 요약 문서를 저장합니다.

## 폴더 구조

- `YYYYMMDD/`: 날짜별 세션 폴더
  - `INDEX.md`: 해당 날짜 세션 요약
  - 구현 완료 보고서들
  - 세션 종료 문서
- `CMIS_VX_*.md`: 버전별 릴리스 노트

## 최신 세션

최신 세션 정보는 [INDEX.md](./INDEX.md)를 참조하세요.

## 참고

세션 문서는 다음 규칙으로 작성됩니다:
- 세션 요약: `SESSION_[날짜]_*.md`
- 구현 완료: `[기능]_COMPLETE.md`
- Phase 보고: `[엔진]_PHASE[N]_COMPLETE.md`
- 분석 문서: `[주제]_ANALYSIS.md`
