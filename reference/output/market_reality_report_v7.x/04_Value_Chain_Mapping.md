## 가치사슬 맵핑 개요

### 관찰 원칙
1. **돈의 흐름**: 누가 → 누구에게 → 얼마를 지불하는가
2. **가치사슬 단계**: 원재료/콘텐츠 → 중간 단계 → 최종 소비자
3. **마진 관찰**: 각 단계에서 얼마를 가져가는가

### 데이터 출처
- ✅ 확정 데이터: SRC-ID 참조
- 🔍 추정 필요: EST-ID 부여 (Estimator 협업)
- 📊 업계 관행: 일반적 마진율 (추정)

## BM별 가치사슬 맵핑

### BM-01: 오프라인 종합 학원

#### 가치사슬 구조

```
[학습자] → [학원] → [강사] + [교재] + [시설]
          (100%)   (40-50%) (10-15%) (20-30%)
```

#### 돈의 흐름 상세

**1단계: 학습자 → 학원 (수강료 지불)**
- **거래 금액**: 월 30-60만원 (3개월 수강권 기준: 90-180만원)
- **결제 방식**: 선불 (수강권 선구매)
- **거래 빈도**: 3-6개월 주기

**2단계: 학원 → 강사 (급여 지급)**
- **지급 형태**: 월급 or 시간당 강의료
- **마진율 추정**: 학원 매출의 40-50% (강사 인건비)
- **근거**: 일반적 교육 서비스업 인건비 비중
- **EST-ID**: EST_20241204_010 (추정 필요)

**3단계: 학원 → 교재 공급사**
- **지급 형태**: 교재 구매 (학습자 직접 구매 or 학원 포함)
- **마진율 추정**: 매출의 10-15%
- **EST-ID**: EST_20241204_011

**4단계: 학원 → 시설/임대료**
- **지급 형태**: 월 임대료, 관리비
- **마진율 추정**: 매출의 20-30% (강남 등 주요 상권)
- **EST-ID**: EST_20241204_012

**5단계: 학원 순이익**
- **추정 순이익률**: 10-20%
- **EST-ID**: EST_20241204_013

#### 관찰된 패턴
- 선불 수강권 모델 → 학원 현금흐름 우위
- 강사 인건비 비중 높음 → 강사 교섭력 중요
- 주요 상권 입지 → 임대료 부담 높음

#### 예시: YBM어학원 (SRC_20241204_001)
- **2023 매출**: 817억원 (전체, 어학 부문 분리 불가)
- **추정 어학 매출**: 500-600억원 (EST-ID 필요)
- **추정 순이익**: 50-120억원 (10-20% 가정)
