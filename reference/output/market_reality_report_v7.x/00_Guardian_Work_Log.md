## 로그 개요

### 프로젝트 목표
한국 성인 어학교육 시장의 구조를 **Collectively Exhaustive**하게 파악하고, BM별/도메인별 분류, Top-N 플레이어, 가치사슬, 시장규모, 경쟁강도, 교섭력을 **완전히 추적 가능한 방식**으로 분석

### 핵심 원칙
1. **MECE**: 모든 분류가 상호 배타적이고 전체 포괄적
2. **완전한 추적성**: 모든 주장에 SRC-/EST-/ASM- ID 부여
3. **검증 게이트**: 주요 결론은 3명 검증자 통과 필수
4. **Needs 중심**: 시장을 특정 BM/도메인으로 한정하지 않음

### 품질 기준
- 결과물 퀄러티가 최우선
- 필요한 모든 작업에 최대 시간과 자원 투입
- 깊이 있고 자세한 분석
